# Contributing to VAANI

Thank you for your interest in contributing to VAANI! We welcome contributions from everyone.

## How to Contribute

### 1. Reporting Bugs
If you find a bug, please create a GitHub Issue including:
- A description of the bug.
- Steps to reproduce it.
- Your browser and OS version.

### 2. Suggesting Features
We love new ideas! Open an issue with the tag `enhancement` to discuss your idea before writing code.

### 3. Submitting Pull Requests
1. Fork the repo.
2. Create a new branch.
3. Make your changes.
4. Ensure the app runs without errors (`npm run dev`).
5. Submit a PR!

Thank you for helping us build an accessible future!
